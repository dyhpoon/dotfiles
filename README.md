##shortcut keys
```^e,^y``` scroll up & down
```^f,^b``` scroll up & down one page
```H,M,L``` move cursor to the top, mid, bot of the window
```f, ;``` search char, and move to the next occurence
```F``` backward of f
```C``` c$
```s``` cl
```S``` ^C
```I``` ^i
```A``` $a 
```diw``` - ```caw```
```di)``` delete all within parenthesis
'''va"``` visually select everything within doublequotes
